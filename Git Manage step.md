# Git Manage step



1\.  git pull : 연동시킨 서버에 있는 모든 파일의 내용을 불러드린다.

2\. git add ./\* : 모든 파일을 관리하겠다고 선언

3. git status : 변화(상태)를 확인.

&#x09;2.1. 변화가 있다면 : "Modifile"

&#x09;2.2. 변화가 없으면 : "Nothing to Commit"

4. git commit -m "Mesagge" : 작업 메시지 작업 생성 (수정한 정보들이나 파일등)

5. git push : Github에 파일 전송을 해서 연동시킴.
